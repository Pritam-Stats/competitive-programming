
# Competitive Programming Architect

A centralized hub for competitive programming (CP), contest up-solving, and algorithmic research. This repository tracks my journey through diverse platforms, focusing on computational efficiency, mathematical logic, and systematic problem-solving.

## Repository Structure

The organization follows a platform-first hierarchy to manage practice sheets and live contest rounds effectively:

* **/codeforces**: 
    * `level-800-cp31/`: Progress on the CP-31 beginner problem sheet consists of 31 800 level problems from CodeForces (By TLE Eliminators)
    * `contests/`: Solutions from live and virtual rounds (Div. 2, Div. 3, etc.). (pushed after the contest ends)
* **/atcoder**: Focused on AtCoder Beginner Contests (ABC) and high-speed logic puzzles.
* **/cses**: Implementation of the standard CSES Problem Set—the gold standard for foundational algorithms.
* **/foundations**: Deep dives into data structures and system-level logic.
* **/templates**: Optimized boilerplates for C++ and Python to minimize implementation time.

> **Architecture Note:** LeetCode solutions are maintained in my `Dsa_with_Python` repository to keep interview-specific preparation distinct from performance-based competitive coding.
